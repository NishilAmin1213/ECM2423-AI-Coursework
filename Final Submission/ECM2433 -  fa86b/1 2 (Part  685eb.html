Extra Features

  • GUI for easy to use application of the A* Algorithm and its Heuristics for the 8-Puzzle
  • Error Catching for an invalid input board
  • Provides data from the running of the code
    • No. of moves required to solve the problem
    • No. of outer while loops used during execution
    • Exact swaps required to solve the problem manually
  • Dropdown menu for selecting a heuristic (Manhattan (Auto), or Hamming)

Error Catching

The Code is able to catch multiple types or error

  • If there are repeated numbers in the input of goal boards
  • If there are unpopulated tiles in the input or goal boards

If there is an issue with the input, an error message is displayed, when pressing the OK button, the code is restarted (This only works when the code is run within an IDE)
