ReadMe

Name

“Nishil’s Evolutionary Algorithm to Solve Sudoku Puzzles”

Description

This project uses my own implementation of an Evolutionary (Genetic) Algorithm to attempt to solve Sudoku puzzles. This would take in a grid, and use the genetic algorithm to find a solution where the grid is full of correct values and the rules of sudoku have been followed

Installation/Running

Please Note, in this version, an empty MUST be represented by a ‘0’ if modifying the input board in the code

Prerequisites → the library NUMPY is used in this version, and it will require installation into the environment prior to the running of the code

To run the program, the main.py file should be run, and this will carry out all the setup and program execution automatically. Please note that the other required python files(board.py) must be in the same directory as main.py in order for the code to run.

CHANGABLE VARIABLES

The algorithm used for the initialisation of the population can be chosen by setting the constant named “FILL” at the top of board.py, this should be set to either “random” or “weighted”

The algorithm used for the selection of the parents can be chosen by setting the constant named “SELECTION” at the top of main.py, this should be set to either “fittest” or “random"

  • A wider range of Selection, Crossover and Initialisation algorithms
